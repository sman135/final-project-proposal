Spence's Cookie Clicker

Team: Spencer Kolman (solo)

Last Update: 11-8-2020 (sprint 5)

Description

Spence's Cookie Clicker is a simple game that allows you to click a button to add "cookies" to your collection. It also allows you to use power-ups to make each click more powerful to add more cookies per click.

Desired Features

  • Auto-clicking
  • Increases to the value of a click
  • Ability to prestige (default click becomes more powerful)

Additional Features

  • Animations
  • Detailed graphics
  • Ability to save progress

Languages and Platforms

Spence's Cookie Clicker uses python, Tkinter, and system libraries.

Technical Features Implemented

Features we learned in class:

  • Loops
  • Functions
  • Variables

Features that go beyond what we learned in class:

  • Utilizes Tkinter
  • Buttons (that can be clicked and perform functions)
